# Spec: Psyche Screen — Live Lain State Visualizer (Issue #19)

## Problem
Psyche screen shows raw JSON. Needs PSX-aesthetic UI showing Lain's inner state.

## Data Source
GET /api/psyche — returns Lain's STATE.yaml: tasks, wave_status, open_prs, notes, metrics

## Implementation
- frontend/js/psyche.js — new file, fetch /api/psyche on init + every 30s
- Parse state.tasks[0] — current task
- Parse wave_status strings → dot-bar rendering
- Parse open_prs → list with Pr001, Pr002 codes
- Parse notes → last 3 lines
- Parse metrics object → summary line
- On update: flash screen (glitch class for 200ms)
- frontend/index.html — add psyche screen div (id="screen-psyche"), add psyche.js script
- frontend/css/style.css — add .psyche-section, .wave-dots, .pr-item styles (PSX aesthetic)

## PSX Aesthetic Details
- Section headers: orange (#ff8c00), uppercase, Share Tech Mono
- Values: cyan (#00d4aa)
- Codes (Lda077, Pr001): green monospace (#00ff88)
- Dots: filled ● = orange, empty ○ = dark gray
- Lines: orange horizontal rules
- Font size: small (VT323 or Share Tech Mono)

## Navigation
- Hub nav item "PSYCHE" → navigates to screen-psyche
- Back button / Esc → back to hub
- screen-psyche uses same screen transition system as other screens

## Acceptance
- Psyche screen shows real data from /api/psyche
- Updates every 30s
- Looks like PSX data terminal, not JSON
- Wave status renders as dot-bars
