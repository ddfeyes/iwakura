# Spec: PSYCHE Mood Indicator (Issue #36)

**Date:** 2026-03-16  
**Issue:** #36  
**Priority:** P1

## Summary
Add a mood indicator to the PSYCHE screen that algorithmically derives Lain's emotional state from observable activity signals (think cycles, tasks, messages, errors).

## Backend Changes

### `backend/main.py` — extend `/api/psyche`

Add a `_derive_mood()` helper function called inside `api_psyche()`.

**Input signals (all from existing data already loaded):**
- `state.tasks` — count in_progress tasks, count failed tasks
- `think` / `think_delta` — recent think cycles, delta count
- `initiative` — messages_sent_today, last_message_at (recency)
- Look at diary persistence (localStorage is frontend-only, skip)
- Check `state.notes` for recent timestamps (within last 30 min)

**Mood derivation logic:**
```python
def _derive_mood(state, initiative, think, think_delta):
    signals = []
    score = 0  # weighted score → maps to mood
    
    # Active tasks
    active_tasks = [t for t in state.get("tasks", []) if t.get("status") == "in_progress"]
    if active_tasks:
        score += 30
        signals.append(f"{len(active_tasks)} task(s) active")
    
    # Recent think activity
    if think_delta and think_delta.get("cycles", 0) > 0:
        score += 25
        signals.append(f"think cycles: {think_delta['cycles']}")
    
    # Recent initiative messages
    msgs_today = initiative.get("messages_sent_today", 0)
    if msgs_today > 3:
        score += 20
        signals.append(f"{msgs_today} messages sent today")
    elif msgs_today > 0:
        score += 10
        signals.append(f"{msgs_today} message(s) today")
    
    # Error signals
    failed = [t for t in state.get("tasks", []) if t.get("status") == "failed"]
    if failed:
        score -= 20
        signals.append(f"{len(failed)} failed task(s)")
    
    # Map score to mood
    if score >= 60:
        return {"label": "FOCUSED", "intensity": min(score, 100), "signals": signals, "color": "#ff8c00"}
    elif score >= 35:
        return {"label": "RESONANT", "intensity": score, "signals": signals, "color": "#00ff88"}
    elif score >= 15:
        return {"label": "DEEP", "intensity": score, "signals": signals, "color": "#00d4aa"}
    elif failed:
        return {"label": "ALERT", "intensity": max(30, abs(score)), "signals": signals, "color": "#ff4444"}
    else:
        return {"label": "IDLE", "intensity": 10, "signals": signals or ["no recent activity"], "color": "#8b7cc8"}
```

Add to response: `data["mood"] = _derive_mood(data.get("state",{}), data.get("initiative",{}), data.get("think",{}), data.get("think_delta",{}))`

## Frontend Changes

### `frontend/js/psyche.js`

In `_render(data)`, extract `mood` from data and call `this._renderMood(mood)` to prepend it to html.

```js
_renderMood(mood) {
    if (!mood) return '';
    const { label, intensity, signals, color } = mood;
    const bars = Math.round(intensity / 10);
    const barHtml = '█'.repeat(bars) + '░'.repeat(10 - bars);
    return `
        <div class="mood-indicator" style="--mood-color:${color}">
            <div class="mood-header">
                <span class="mood-pulse"></span>
                <span class="mood-label" style="color:${color}">${label}</span>
                <span class="mood-intensity">${intensity}%</span>
            </div>
            <div class="mood-bar">${barHtml}</div>
            <div class="mood-signals">${signals.map(s => `<span>${s}</span>`).join('')}</div>
        </div>
    `;
}
```
