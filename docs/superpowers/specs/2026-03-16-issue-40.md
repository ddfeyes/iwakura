## Backend Changes (`backend/main.py`)

### New endpoint: `GET /api/memory/search?q=<query>`
```python
@app.get("/api/memory/search")
async def api_memory_search(q: str = ""):
    if not q or len(q) < 2:
        return JSONResponse({"results": []})
    query = q.lower()
    results = []
    for f in sorted(LAIN_MEMORY.iterdir()):
        if not f.is_file():
            continue
        try:
            content = f.read_text(errors="replace")
            count = content.lower().count(query)
            if count == 0:
                continue
            # Find first match for excerpt
            idx = content.lower().find(query)
            start = max(0, idx - 60)
            end = min(len(content), idx + len(query) + 80)
            excerpt = content[start:end].replace('\n', ' ').strip()
            results.append({"name": f.name, "excerpt": excerpt, "match_count": count})
        except Exception:
            pass
    results.sort(key=lambda x: x["match_count"], reverse=True)
    return JSONResponse({"results": results[:8], "query": q})
```

## Frontend Changes (`frontend/js/memory.js`)

### In `_load()` after list renders: bind search
### New `_bindSearch()` method:
- `input` event → `_filterList(val)`: filter `this._files` by name, re-render sidebar
- `keydown Enter` → `_contentSearch(val)`: call `/api/memory/search?q=val`, show results
- `keydown Esc` → `_clearSearch()`: restore full list

### `_filterList(q)`:
- Filter `this._files` by `f.name.toLowerCase().includes(q.toLowerCase())`  
- Re-render sidebar with filtered files
- Update badge: `[N FILES]` in dim green

### `_contentSearch(q)`:
- Show loading in sidebar
- Fetch `/api/memory/search?q=q`
- Render results with name + excerpt snippet
- Badge: `[N MATCHES]` in orange

### `_clearSearch()`:
- Clear input, hide badge, restore full list

### In `renderMemContent()`: highlight search term
- After rendering, replace occurrences of query with `<mark class="mem-kw-match">...</mark>`
