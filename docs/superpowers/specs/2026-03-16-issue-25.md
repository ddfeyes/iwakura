# Spec: Ambient Audio System — Issue #25

**Date:** 2026-03-16  
**Issue:** #25 — feat: ambient audio system — drone hum + UI sounds  
**Priority:** P1

## Summary
Extend existing `audio.js` with a full Web Audio API ambient system. No external files — all sounds synthesized in-browser.

## Requirements

### Ambient Drone
- Low-frequency oscillator (40–80 Hz) with a slight LFO wobble (0.1–0.3 Hz, ±2 Hz)
- Filtered through BiquadFilter (lowpass, Q=2) for warmth
- Very subtle volume (gainNode ~0.05)
- Starts on first user interaction (click/keypress), loops forever

### Boot Sound
- On app boot: short static burst (white noise, 0.3s, fast attack/decay)
- Followed by a low sine sweep 80→40 Hz over 1s

### Navigation Click
- Soft sine tick: 600 Hz, 0.05s, very low gain (0.1)
- Triggered on every screen switch

### Screen Transition Glitch
- White noise burst (0.1s) + pitch-shifted click
- Triggered on screen enter animations

### Chat Message Beep
- 880 Hz sine, 0.08s, gain 0.08
- Triggered when new message received

### Volume Control
- Slider already exists in UI (wire to master GainNode)
- Persists in localStorage key `iwakura-audio-volume` (default 0.7)

### Mute Toggle
- Mute button in footer/status bar
- Persists in localStorage key `iwakura-muted`
- Mute = set master gain to 0 (not suspend context — faster unmute)

## Implementation Plan

1. **audio.js** — complete rewrite/extension:
   - `IwakuraAudio` class (or extend existing)
   - `init()` — sets up AudioContext + master gain + drone chain; deferred to first interaction
   - `playBoot()`, `playClick()`, `playGlitch()`, `playBeep()` methods
   - `setVolume(v)`, `toggleMute()` — persist to localStorage
   - Export as `window.audio` for app.js access

2. **app.js** — wire hooks:
   - `audio.playClick()` on screen switch
   - `audio.playGlitch()` on screen transition
   - `audio.playBeep()` on new message (if psyche/memory updated)

3. **index.html** — ensure mute button has `id="mute-btn"` and volume slider `id="volume-slider"`

4. **style.css** — minor: active state for mute button
