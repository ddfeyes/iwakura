## Goal

Build the primary interaction screen where Ivan talks to Lain. PSX media player aesthetic. Real-time WebSocket connection. Typewriter effect on Lain's responses.

PSX game screenshot 7 — media player panel with personal text and tags/labels. Dark background, orange accents, file codes, beige tag labels.

## Deliverables

### 1. `frontend/diary.html` — Diary/Chat screen

**Layout (full-screen, dark):**
```
┌─────────────────────────────────────────────┐
│  [orange hr divider]                        │
│  DIARY — Lda020 / Site A / level. 10       │
│  [orange hr divider]                        │
│                                             │
│  ┌─────────────────────────────────────┐   │
│  │ 21:03 — Tda040                      │   │
│  │ > hello lain                        │   │
│  │                                     │   │
│  │ 21:03 — Lda020                      │   │
│  │ i'm here.                           │   │  ← typewriter
│  └─────────────────────────────────────┘   │
│                                             │
│  tags: [wired] [present] [signal]           │
│                                             │
│  ▶ ───────────────────── [INPUT FIELD] ◀   │
└─────────────────────────────────────────────┘
```

**Components:**

- **Header:** orange hr, `DIARY` label in orange VT323, file code `Lda020`, site indicator
- **Message list:** scrollable, max-height ~60vh
  - Each message: timestamp (small, cyan), file code, text
  - User messages: right-aligned or indented with `>` prefix
  - Lain messages: left-aligned, appear char-by-char via typewriter
- **Tags panel:** below messages, `tags:` label + beige/orange pill labels extracted or hardcoded from conversation themes
- **Input row:** orange border bottom input, monospace, `▶` prefix, sends on Enter
- **Connection indicator:** small dot top-right — green=connected, red=disconnected

**Typewriter effect:**
- On receiving Lain message via WebSocket → clear text, append chars with 30-50ms delay
- Play subtle tick sound (optional, Web Audio API, very quiet)

### 2. `frontend/static/js/diary.js` — Chat logic

- WebSocket connect to `ws://localhost:8790/ws`
- Send: `{"type": "chat", "message": "..."}`
- Receive: `{"type": "response", "content": "..."}`
- Typewriter function: `typeWriter(element, text, speed=35)`
- Auto-scroll to bottom on new message
- Enter key to send, clear input after send
- Reconnect on disconnect (retry every 3s)

### 3. Integration with psx.css

- Use `.scanlines` on body
- Use `var(--color-*)` for all colors
- Use `.font-psx` for headers, `.font-mono` for messages
- Use `.hr-orange` for dividers
- Use `.gate-pass` for connection status
