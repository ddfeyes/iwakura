## Summary
Add a 5th navigation node "TASKS" to the Iwakura dashboard displaying Lain's live task state from STATE.yaml. Auto-refreshes every 15s.

## Backend

### GET /api/tasks
- Read `~/agents/lain/STATE.yaml` (same path as /api/psyche reads it)
- Parse `tasks` array from YAML
- For each task, return:
  - `id`, `goal`, `status` (RUNNING/PAUSED/DONE/PENDING)
  - `progress`: `{ done: N, remaining: N, pct: float }` computed from `remaining[]` and `stats.tasks_done`
  - `recent_done[]`: last 3 items from done list (if available)
  - `remaining[]`: next 3 items from remaining list
  - `stats{}`: raw stats block
- Also return top-level metrics:
  - `active_count`, `done_count`, `paused_count`
  - `tests_passing` (from stats if available), `prs_merged`, `features_shipped`
- Return `{ tasks: [...], metrics: {...} }`
- Handle missing/malformed YAML gracefully (return empty state)
