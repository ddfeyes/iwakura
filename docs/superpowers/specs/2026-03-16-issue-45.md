# Spec: DIARY — Message History Persistence + Session Restore (Issue #45)

## Problem
On page reload, all diary messages are lost. OpenClaw session ID is preserved but the visual chat history is gone.

## Solution

### Backend (backend/main.py)

1. Add a diary_history store: `DIARY_HISTORY_FILE = Path(__file__).parent / "diary_history.json"`
   - Format: list of dicts: `{role: "user"|"lain", text: str, code: str, timestamp: str}`
   - Max 200 entries (rotate oldest when exceeded)

2. Helper functions:
```python
def load_diary_history() -> list[dict]:
    try:
        if DIARY_HISTORY_FILE.exists():
            return json.loads(DIARY_HISTORY_FILE.read_text())
    except Exception:
        pass
    return []

def save_diary_entry(entry: dict):
    h = load_diary_history()
    h.append(entry)
    if len(h) > 200:
        h = h[-200:]
    DIARY_HISTORY_FILE.write_text(json.dumps(h, ensure_ascii=False))
```

3. New endpoint:
```python
@app.get("/api/diary/history")
async def api_diary_history():
    return load_diary_history()[-50:]  # last 50 messages
```

4. In the WebSocket handler (ws_chat), after user message received and after Lain response received, call `save_diary_entry()`:
   - User: `{"role": "user", "text": msg_text, "code": gen_file_code(), "timestamp": utcnow().isoformat()+"Z"}`
   - Lain: `{"role": "lain", "text": full_response_text, "code": gen_file_code(), "timestamp": utcnow().isoformat()+"Z"}`
   - For streaming, accumulate tokens and save at end of stream

### Frontend (frontend/js/chat.js or diary.js)

On DIARY screen init, before starting WebSocket:
1. Fetch GET /api/diary/history
2. If history has entries, insert a separator: `<div class="history-separator">── PREVIOUS SESSION ──</div>`
3. Render each historical message with class `msg-historical` (dim opacity: 0.5, no typewriter effect)
4. Auto-scroll to bottom after history loads
5. Then open WebSocket for new messages (existing flow)

CSS for historical messages:
```css
.msg-historical { opacity: 0.45; filter: saturate(0.5); }
.history-separator { text-align:center; color: var(--cyan); opacity: 0.4; font-size:10px; margin: 12px 0; letter-spacing: 2px; }
```

## Files
- `backend/main.py` — add history file, endpoint, save on WS messages
- `frontend/js/chat.js` or `diary.js` — load history on init
